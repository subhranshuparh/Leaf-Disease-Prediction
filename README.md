A deep learning–based web application to detect plant leaf diseases from images using VGG16 transfer learning and Streamlit.

This project helps identify plant diseases early, supporting farmers and agriculture researchers.

Features:

Detects plant leaf diseases from images

Uses VGG16 (Transfer Learning)

Fast inference with pretrained CNN

Upload image & get prediction instantly

User-friendly Streamlit web app

Shows prediction confidence

Model Details:

Architecture: VGG16 (ImageNet pretrained)

Type: CNN (Transfer Learning)

Input Size: 224 × 224 RGB image

Output: Disease class + confidence

Training Platform: Google Colab

Inference Platform: VS Code + Streamlit

Only the weights are loaded during deployment for stability.
